# Frontend Quiniela Conde

## Estructura base
- src/assets/img -> imágenes y logo
- src/css -> estilos globales
- src/js -> scripts globales
- src/views/auth -> vistas de autenticación
- src/components -> componentes reutilizables
- src/services -> llamadas al backend
- src/utils -> helpers

## Nota
Coloca tu logo personal en:
`src/assets/img/logo.png`
